## Sobre o Projeto

No projeto semestral, contruímos um site informativo, relacionado na área de jogos. 

Na parte do frontend utilizamos o framework **bootstrap** para construir um site responsivo. Utilizamos classes do bootstrap para a construção do menu bar, do carrossel, dos cards, dos modals e das inserções dos ícones.

O uso do JavaScript foi para guardar os links do menu bar em um ferramenta de um dropdown list (lista supensa). O **bootstrap** não foi utilizado em todos os arquivos, apenas nos arquivos que continham muitas informações.

Na área do usuário, que é a parte de cadastro e do login, foi utilizado a linguagem PHP para conversar com o banco de dados. O usuário quando cadastrado, e depois de ser logado, um aviso será emitido, mostrando que o usuário está logado na seguinte sessão.

No PHP, há dois arquivos, que tem como função de inserir os dados cadastrado do usuário e de consultar a existência do mesmo.

Foi utilizado o adobe XD para criar o layout princiapal do site, o <code>index.php</code>, e o mesmo modelo utilizei pro <code>servico.html</code> e <code>explorar.html</code>. O ícone do site que é um controle de video game utilizamos, o adobe SPARK para sua criação.

Em algumas partes do site, decidimos utilizar um liguagem mais informal, por conta do conteúdo do site ser algo um pouco mais moderno.

E o código do banco de dados, estará disponível no arquivo <code>teste.sql</code>.
